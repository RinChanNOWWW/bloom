# blooming

BT/PT 站 RSS 更新通知器。

## 支持的站点

- [蜜柑计划](https://mikanani.me/)
- [BYRBT](https://byr.pt/)
- [北洋园PT](https://tjupt.org/)

## 通知方式

- QQ 机器人: [go-cqhttp](https://github.com/Mrs4s/go-cqhttp).

## 使用

```bash
blooming -c config.toml

# or

blooming -c config.toml -d # daemon mode
```

配置文件请参考 [config.toml](examples/config.toml).
